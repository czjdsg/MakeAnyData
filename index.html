User-Intent-Driven Data Construction

DataClaw: Make ANY Data You Want

Turn raw inputs into user-intent-aligned, trainable multimodal data.

DataClaw is built around a simple premise: users do not want a fixed dataset recipe. They want the exact data form that matches their downstream goal. From videos, images, GUI traces, robot trajectories, and more, DataClaw infers intent and composes the right trainable output.

Method

Two agents turn one raw video into the exact data the user wants.

DataClaw first interprets the user intent, then fans out into one or more specialized synthesis models. If the original request is broad, multiple task types remain alive and are synthesized in parallel instead of being collapsed into a single schema.

Inputs

Raw material + user goal

Raw Video

User Intent

Evaluation Logic

DataClaw is evaluated by what matters for data construction.

The core question is not whether outputs look plausible in isolation. The core question is whether the system constructs the right data form for the right user intention, and whether that data is actually useful for training.
